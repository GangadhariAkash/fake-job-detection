# 💼 Fake Job Detection System

This project uses Machine Learning and NLP to detect fraudulent job postings.

## 🚀 Features
- Detects Fake vs Real Jobs
- Confidence Score
- Risk Level Analysis
- Company Type Detection
- Streamlit Web App

## 🛠️ Tech Stack
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## ▶️ How to Run
```bash
streamlit run app.py
